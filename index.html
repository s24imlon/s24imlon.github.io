Projects

Red-teaming OpenAI

This project leveraged OpenAI's API to generate and test adversarial prompts designed to elicit harmful responses from language models. Multiple LLMs were evaluated using an automated classifier to analyze their susceptibility and response patterns. The findings provided insights into jailbreak effectiveness, model robustness, and safety mechanisms.

Detecting patronising and condescending language using transformers

This project involved developing a binary classification model to detect patronizing and condescending language in text. Using transformer-based models, various techniques such as data augmentation, hyperparameter tuning, and ensembling were explored to improve performance beyond the RoBERTa-base baseline. The results were analyzed to understand the impact of input length, data categories, and model biases.

Fraud Network Analytics

Developed an AI-powered risk assessment tool that mapped the digital footprint of bad actors to identify potential fraudulent activities. Improved the Financial Crime and Compliance team's efficiency by 50%, enabling faster detection of high-risk customers. Used graph-based AI models to uncover hidden relationships in financial transactions.

Time Deposit Pricing

Designed and executed A/B experiments to analyze customer price sensitivity and deposit behavior. Developed a Time Deposit Pricing Engine, aligning deposit pricing with market interest rate trends and customer behavior, leading to a 15% revenue increase.

Multi-modal Knowledge Distillation

This project explores knowledge distillation techniques to create efficient, multi-modal machine learning models for deployment on resource-constrained edge devices. By transferring knowledge from a complex teacher model, the goal is to develop a compact student model capable of processing diverse sensor inputs—including audio, images, accelerometer, and health data—while maintaining accuracy. The research focuses on optimizing model compression, computational efficiency, and adaptability for TinyML applications.
